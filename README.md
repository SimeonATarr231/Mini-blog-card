# Minimal Blog Card

A clean and modern mini blog card built using only HTML and CSS.  
This project is designed to highlight light images with calm visuals, soft spacing, and simple typography.

## Preview
The card features:
- A light themed image at the top
- A clear title and short description
- Rounded corners and subtle shadow for a modern look

## Purpose
This project was created to practice:
- Basic HTML structure
- CSS layout and spacing
- Card based UI design
- Clean and minimal visual presentation

It is suitable for beginners who want to understand how simple components are built for blogs, portfolios, or landing pages.

## Technologies Used
- HTML5
- CSS3

No frameworks or libraries were used.

## How to Use
1. Clone or download the repository
2. Open `index.html` in your browser
3. Replace the image source with your own light themed image
4. Customize the text to fit your content
